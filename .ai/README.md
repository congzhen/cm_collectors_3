# .ai 工作区

`.ai/` 是本仓库的 AI 协作工作区，用于存放面向 Codex、开发代理和维护者的项目资料。

## 目录约定

- `docs/`：稳定项目资料，按主题拆分维护。
- `docs/INDEX.md`：文档总索引，新增文档后应同步更新。
- `docs/architecture/`：项目地图、模块职责、系统边界。
- `docs/development/`：开发命令、构建、测试、提交约定。
- `docs/design/`：功能设计、技术方案。
- `docs/testing/`：测试计划、验收清单。

根目录 `agents.md` 是 AI 工作入口，只保留长期稳定的协作规则和文档索引；较长项目说明应迁移到本目录。
